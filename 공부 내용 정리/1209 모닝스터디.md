
## 반복문 + 함수 이용한 [회문 판별]
### 순서대로 정리
- 조건: 문자열을 입력 받아서 회문이면 true 반환, 아니면 false 반환

#### 1. 방식
2개의 방식 ex. level 단어
**1 -1.** 문자열을 한글자씩 새로운 리스트에 담아서 뒤집은 문자의 리스트와 기존 리스트 비교
=> level 한글자씩 5번 + 뒤집은 문자열 5번 **총 10번**

**1 -2.** 문자열 한글자씩 리스트에 담아서 리스트 반복하면서 첫글자와 끝글자 비교
=> level 앞 뒤로 비교 시에 2번 반복해서 **길이 총 5번 필요**

[참고사항]
- 공간복잡도: 메모리 공간을 얼마나 차지하느냐 (변수를 얼마나 선언했냐)
- 시간복잡도: 시간이 얼마나 걸렸냐 (반복문을 얼마나 사용했냐)

=> 두번째 방식이 효율적으로 두번째 방법으로 문제 풀이

---

#### 2. 문제 풀이 순서
**2 -1.** 참인지 거짓인지 판별 함수
``` bool ```
**2 -2.** ``` list ``` 담아주기
**2 -3.** ``` for ``` 문으로 반복문
**2 -4.** ``` if ``` 문으로 조건
**2 -5.** 참인지 거짓인지 출력

---

#### 3. 코드 작성
```dart
bool isPalindrome(String str){
    List<String> charList = str.split('');
    int halfIndex = charList.lenth ~/ 2;
    for (var i = 0; i < halfIndex; i++){
        String char1 = charList[i];
        int endIndex = charList.length - 1 - i;
        String char2 = charList[endIndex];
        if (char1 != char2){
            return false;
        }
    }
    return true;
}

void main(){
    print(isPalindrome("level"));
}
```
