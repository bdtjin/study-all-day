## 이 외 참고 사항

### `-`
특정 단어 제외
ex. dart 라이브러리 파일 가져오기 -python
(이거 안 해도 지금은 dart 치면 다 나옴)

### `site:`
사이트 내 검색
특정 사이트 내에서만 검색할 때 사용
NavMesh site:docs.unity3d.com

### `filetype:`
파일 형식 검색
특정 파일 형식(PDF, PPT 등)만 검색할 때 사용

### `intitle:`
제목에서 검색
웹페이지 제목에 특정 단어가 포함된 결과를 찾을 때 사용
intitle:게임 개발 사업

### `inurl:`
URL에서 검색
URL에 특정 단어가 포함된 결과를 찾을 때 사용
게임 개발 사업 inurl:tutorial

### 고급 검색

### 장단점

**장점:** 여러가지 예시를 볼 수 있고, 응용방법까지 유추할 수 있고 검색을 하면서 새로운 사실들을 많이 알 수 있다. 
다른 사람의 코드를 보고 직접 내가 짜논 코드에 적용시켜야 하기 때문에 코드 사용 후 변화하는 과정을 알 수 있음.
